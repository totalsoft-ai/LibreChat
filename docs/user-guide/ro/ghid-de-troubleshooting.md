# Ghid de troubleshooting

**Eroare la logare**

Contactați **suport tehnic** pentru verificare și rezolvare. Sistemul de verificare a parolei este nefuncțional.

![image](/help-images/ro_0087.png)

**Eroare de procesare Document Loader**

Anumite namespace-uri care nu pot fi definite. Contactati suport tehnic pentru verificare.

![image](/help-images/ro_0088.png)

**Eroare incarcare documente**

A aparut o eroare la incarcarea documentului deoarce contine nume personale, nu pot fi adaugate date personale.

“The document contains personal data and cannot be loaded into the "saft" namespace due to privacy restrictions.”

![image](/help-images/ro_0089.png)

**Eroare conexiune la server**

Incearca reincarcarea paginii si verifica conexiunea la internet.

![image](/help-images/ro_0090.png)
