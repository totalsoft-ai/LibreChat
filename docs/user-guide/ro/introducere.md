# Introducere

**Tessa** este o platformă inteligentă de conversație bazată pe AI concepută pentru a oferi utilizatorilor o interfață modernă și intuitivă de chat. Aplicația integrează asistenți virtuali complementari si o functionalitate de creare a propriilor agenti care genereaza raspunsuri in baza documentiilor incarcate.

Primul asistent funcționează ca un sistem QnA specializat pentru Charisma, oferind răspunsuri la întrebări bazate pe documentația incarcata in Confluence. Utilizatorul formulează întrebări în chat, iar asistentul returnează răspunsuri relevante și contextualizate, acolo unde informația este disponibilă.

Al doilea asistent dispune de o suită de instrumente și funcționalități care direcționează automat solicitările către agenți specializați, oferind o experiență similară cu ChatGPT. Asistentul oferă:


| Agent | Acțiuni | Cuvinte cheie de activare |
| --- | --- | --- |
| Chat | Conversație generală, Q&A | Orice solicitare generală |
| Summarizer | Rezumă texte lungi | "summarize", "summary" |
| Translator | Traduce texte/fișiere | "translate to", "translation" |
| DocuFlow | Generează PRD-uri, planuri, diagrame | "prd", "execution plan", "diagram" |
| Code Review | Analizează și îmbunătățește codul | "code review", "analyze code" |


Optiunea de **File Search** ne ofera posibilitatea sa cerem unui agent informatii in baza unei documentii incarcate local intr-un agent personal sau al echipei.
