# TESSA User Guide

Welcome to the TESSA User Guide! This comprehensive documentation will help you make the most of TESSA's powerful features and capabilities.

## What is TESSA?

TESSA is an all-in-one AI conversation platform that brings together multiple AI models in a single, intuitive interface. Whether you're using OpenAI's GPT models, Anthropic's Claude, Google's Gemini, or other providers, TESSA provides a unified experience with powerful features like custom agents, prompt libraries, file handling, and much more.

## Key Features at a Glance

### Multiple AI Providers
- OpenAI (GPT-3.5, GPT-4, GPT-4 Turbo)
- Anthropic (Claude 3 Opus, Sonnet, Haiku)
- Google (Gemini Pro, Ultra)
- Azure OpenAI
- And many more...

### Powerful Tools
- **Agents**: Create custom AI assistants with specialized tools
- **Prompts**: Build a library of reusable prompt templates
- **Files**: Attach documents, images, and code to conversations
- **Export**: Save conversations in multiple formats
- **Search**: Full-text search across all conversations
- **Share**: Create public links to share conversations

### Customization Options
- **Themes**: Dark, Light, or System-matched
- **Languages**: 36 languages supported
- **Settings**: Fine-tune AI model parameters
- **Presets**: Save and reuse model configurations
- **Tags**: Organize conversations your way

### Security & Privacy
- **Authentication**: Multiple login methods including 2FA
- **Permissions**: Control who can access your agents and prompts
- **Data Control**: Export, delete, or archive your data anytime
- **Secure Sharing**: Share conversations with revocable links
