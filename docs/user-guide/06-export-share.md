# Export & Sharing

Learn how to export your conversations in multiple formats and share them through secure links.

## Exporting Conversations

**Why Export?**
Save discussions for offline reference, create documentation, share insights, archive conversations, include in reports/presentations, backup data, process in other applications.

**Available Export Formats:**

| Format | Extension | Best For |
|--------|-----------|----------|
| **Screenshot** | `.png` | Visual sharing, presentations |
| **Text** | `.txt` | Simple backup, plain text processing |

**How to Export:**

1. **Open Export Dialog:**
   Open the conversation → Click the Share/Export button in the top-right corner → **Select Export**

2. **Choose Format:**
   In export dialog → Find **Type** dropdown → Select format (screenshot, text)
3. **Configure Filename:**
   Enter desired name (default: conversation title, auto-sanitized). Don't include file extension (added automatically).


> **Tip:** Filename sanitized automatically (special characters removed, spaces to hyphens).

## Export Formats in Detail

**Screenshot (.png):**
- **Includes**: Visual rendering, formatted messages, avatars, timestamps, model indicators
- **Best for**: Presentations, visual documentation
- **Characteristics**: High-quality PNG, preserves formatting/colors, cannot be edited

**Text (.txt):**
- **Includes**: Plain text messages, user/AI labels, basic metadata
- **Excludes**: Formatting, images (shown as "[Image]"), branches
- **Best for**: Simple backup, text editors, quick reference
- **Characteristics**: Universal compatibility, easy to search


## Sharing Conversations

**Creating Shareable Links:**
(Feature availability depends on configuration)

1. Open conversation → Three dots menu → **Share**
2. Click **Create Link**
4. Copy link to share

**QR Codes:**
Generate QR code for easy mobile sharing

**Revoking Access:**
Find conversation → Click **Share** → **Delete** → Confirm

## Privacy and Security

**Best Practices:**
- Don't share sensitive information publicly
- Use password protection for confidential content
- Review what you're sharing before generating link
- Revoke links when no longer needed


**Data Considerations:**
- Exported files contain full conversation history
- Screenshots visible to anyone with access
- Shared links may be accessible until revoked
