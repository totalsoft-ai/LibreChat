# Settings & Personalization

Customize TESSA to match your preferences, manage your account, and control your data.

## Accessing Settings

1. Click **user icon** or **menu** (top-right)
2. Select **Settings**
3. Navigate between tabs: General, Chat, Speech, Account, Data, Balance, Commands

## General Settings

### Theme Selection

**Available themes:**
- Dark: Dark background, light text
- Light: Light background, dark text
- System: Matches OS theme

Changes apply immediately.

### Auto-Scroll

**When enabled:** Chat scrolls automatically as AI responds
**When disabled:** Manual scroll control for reviewing earlier messages

### Archived Chats

1. Settings > General > **Archived Chats**
2. View, unarchive, delete, or search archived conversations

## Chat Settings

### Font Size

**Sizes:** Small, Normal (default), Large, Extra Large

Settings > Chat > **Font Size** - Updates immediately

### Text Direction

- **LTR** (Left-to-Right): English, Spanish, French, etc.
- **RTL** (Right-to-Left): Arabic, Hebrew, etc.

### Fork Settings

**Fork behavior options:**
- Always show fork options: Modal with 4 fork choices
- Direct fork to new conversation
- Ask each time

Settings > Chat > **Fork Settings**

### Display Preferences

**Show Thinking/Reasoning:**
- Show/hide AI's thinking process (Claude models)

**Save Badge States:**
- Remember dismissed notification badges

## Account Settings

### Avatar Management

**Upload avatar:**
1. Settings > Account > **Avatar**
2. Upload image (JPG, PNG, GIF - recommended 256x256px, max 2-5MB)
3. Crop if needed
4. Save

**Remove avatar:** Click **Remove Avatar** for default

### Display Username in Messages

Toggle to show/hide username above your messages in chat.


## Data Management

### Clear Conversations

**Options:**
- Clear All: Delete all conversations
- Clear Selected: Choose specific conversations
- Clear by Date: Before certain date
- Clear by Endpoint: From specific models

Settings > Data > **Clear Conversations**

> **Warning:** Cannot be undone. Export important conversations first!

### Delete Cache

Clears cached API responses, temporary data, TTS audio, image previews, local storage.

**When to clear:** Low storage, UI issues, after updates, assets not loading

Settings > Data > **Delete Cache** > Refresh page


### Revoke API Keys

Revoke access tokens, OAuth connections, third-party integrations.

**When to revoke:** Security concern, unused integration, suspicious activity

Settings > Data > **Revoke Keys**

### Shared Links

View and manage publicly shared conversations.

**To manage:**
1. Settings > Data > **Shared Links**
2. View list with URLs, dates, view counts
3. Click **Revoke** to unshare (link becomes invalid immediately)

## Best Practices

### Security
- Review shared links regularly
- Revoke unused API keys/OAuth
- Monitor for unexpected activity
- Log out on shared devices

### Privacy
- Manage AI memories
- Limit sharing to what's necessary
- Use private conversations for sensitive work
- Regularly delete unnecessary conversations
- Export and delete sensitive data
- Check agent and tool permissions

### Accessibility
- Use larger font if needed
- Choose high contrast theme
- Enable STT/TTS for voice interaction
- Reduce TTS playback rate for clarity
- Enable auto-scroll for easier reading
- Learn keyboard shortcuts

## Troubleshooting

**Settings Not Saving:**
- Check internet connection
- Clear browser cache
- Try different browser
- Disable browser extensions
- Check browser console for errors

**Theme Not Applying:**
- Refresh page
- Clear browser cache
- Try different browser
- Manually toggle theme again
